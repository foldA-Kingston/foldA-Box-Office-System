## Installation
### Clone

    git clone (repo)
### Setup
Make sure Python3 is installed. Check using:
    
    python3 -V

1. Create & activate virtual environment in QVFT-Brain directory:

        python3 -m venv venv
        source /venv/bin/activate


2. Install required libraries:

        pip install -r requirements.txt  

3.  Start the server:

        python app.py
